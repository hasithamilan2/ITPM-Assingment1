
# SwiftTranslator Automation – ITPM Assignment 1

## Description

This project contains automated **functional** and **UI test cases** for the **SwiftTranslator (Singlish → Sinhala)** web application.  
The automation framework is implemented using **Playwright** to ensure accurate translation behavior, UI consistency, and robustness of the application.

---

## Tools & Technologies

- Playwright  
- JavaScript  
- Node.js  
- npm  

---

## Test Coverage

The automated test suite covers:

- Positive functional test cases  
- Negative (robustness) test cases  
- UI behavior validation  
- Input length types:
  - **S** – Small input  
  - **M** – Medium input  
  - **L** – Large input  

---

## Prerequisites

Before running this project, ensure the following are installed:

- Node.js (v18 or later)
- npm
